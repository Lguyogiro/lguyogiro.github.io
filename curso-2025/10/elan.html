Elan

Elan es una aplicación de múltiples plataformas (Windows, MacOS, y Linux) que se utiliza para transcribir y anotar audio/video.

Cómo instalar

  • Para instalar Elan, visita la página de descarga: https://archive.mpi.nl/tla/elan/download.
  • Elan es un software gratuito y de código abierto, por lo que puedes descargarlo sin costo alguno.
  • Elan es compatible con Windows, MacOS y Linux. Para esta clase los que esan Mac van a descargar la versión de MacOS (aquí), y las personas con Windows (que usan Windows Subsystem for Linux) deberían de descargar la versión para Windows (aquí).
  • Para instalar Elan, simplemente haz doble clic en el archivo descargado y sigue las instrucciones de instalación.
  • Una vez instalado, puedes abrir Elan desde el menú de aplicaciones de tu sistema operativo.

Uso básico

Su función más común/básico (en mi opinión y experiencia) es la segmentación y transcripción de audio.

Una vez instalado Elan, mira este video explicando cómo empezar con el trabajo de transcripción.

Para esta clase podemos trabajar con puro audio, no necesitamos incluir lo del video, o sea el archivo .mp4, vamos a trabajar con puro .wav

Ejercicio

Siguiendo las instrucciones del video de arriba, importa un audio que tienes (por ejemplo de sus trabajos, o simplemnte grabate diciendo algo). Agrega las líneas de transcripción (lo de "traducción" es opcional), y transcribe unos segmentos.

Salida de Elan

Speach

Speach es un paquete de Python que permite trabajar con datos de audio y transcripción en varios formatos.

En esta página, vamos a ver cómo instalarlo y usarlo para leer un archivo .eaf y agreagar una nueva línea. Esto puede ser útil, por ejemplo, si ya tienes la transcripción en una línea y quieres agregar otra línea con la transcripción normalizada.

Instalación

  • Para instalar Speach, abre una terminal y ejecuta el siguiente comando:
    • pip install speach
  • Esto descargará e instalará el paquete Speach y sus dependencias.
  • Una vez instalado, puedes importar Speach en tu código de Python con la siguiente línea:
    • import speach

Uso básico

  • Para leer un archivo .eaf con Speach, utiliza la función read_eaf. Por ejemplo:
    • eaf_file = speach.elan.read_eaf('ruta/al/archivo.eaf')
  • Esto cargará el archivo .eaf y lo almacenará en la variable eaf_file.
  • Una vez que tengas el archivo cargado, puedes acceder a las diferentes líneas (tiers) del archivo y manipularlas según tus necesidades.
  • Para ver las líneas que hay en el archivo: 
    print(eaf_file.tiers())
  • Ver las anotaciones en una línea: 
    eaf_file.tiers['nombre_de_la_linea'].annotations:
  • Para iterar sobre esas anotaciones: 
    for annotation in eaf_file.tiers['nombre_de_la_linea'].annotations:
    print(annotation.start, annotation.end, annotation.text)

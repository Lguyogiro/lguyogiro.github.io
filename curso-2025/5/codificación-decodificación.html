Unicode y la Codificación/Descodificación de texto

En esta página vamos a aprender sobre Unicode y cómo se representa el texto en Python.

Codificación/Descodificación de Texto

  • Para almacenar texto en un archivo o en la memoria de una computadora, necesitas convertir los caracteres en números (1s y 0s).
  • En Python, es fácil imaginar cómo se representa un número, como 23, de esta manera: Sólo se tiene que convertir a binario (10111).
  • Pero, ¿cómo se representa una letra o un símbolo? ¿Cómo se representa la letra "A"?
  • Inicialmente, se utilizó un sistema llamado ASCII (American Standard Code for Information Interchange) que asignaba un número (entre 0 y 127) a cada letra y símbolo.
  • Por ejemplo, la letra "A" se representaba con el número 65.
  • El problema con ASCII es que sólo soporta 128 caracteres, lo cual no es suficiente para representar todos los caracteres de todos los idiomas.
  • La codificación de texto es el proceso de convertir un carácter en un número.
  • Hay varios esquemas de codificación de texto, como ASCII, UTF-8, UTF-16, UTF-32, latin-1, etc.
  • UTF-8 es el esquema de codificación de texto más común y se utiliza en la mayoría de los sistemas y páginas web.
  • UTF-8 es compatible con Unicode y puede representar todos los caracteres Unicode.

Entonces la codificación (encoding) de texto es la manera en la conviertes el texto en binario para almacenar y/o transferir el texto en la computadora

Y la descodificación (decoding) de texto es la manera en la que conviertes el binario de nuevo a texto para que puedas leerlo.

Ejemplo de Codificación y Descodificación en Python

texto = "Hola, ¿cómo estás?"
encoded_text = texto.encode("utf-8")
print(encoded_text)  # b'Hola, ¿cómo estás?'
decoded_text = encoded_text.decode("utf-8")
print(decoded_text)  # Hola, ¿cómo estás?

Cuando escribes y/o lees un archivo con texto, tu sistema operativo tiene que usar una de las varias posibles codificaciones de texto para convertir el texto en binario.

En sistemas Linux/MacOS, la codificación default es UTF-8.

Si lees un archivo usando una codificación diferente a la con la que se escribió, es posible que veas caracteres extraños o incorrectos.

Unicode

  • Unicode es un sistema que asigna un número único a cada carácter, independientemente de la plataforma, el programa o el idioma.
  • Unicode soporta más de 143,000 caracteres, incluyendo letras, números, símbolos y emojis.
  • Por ejemplo, la letra "A" tiene el número Unicode 65 y el emoji "😊" tiene el número Unicode 128522.

En Python, para ver el número Unicode de un carácter, puedes usar la función ord():

print(ord("A"))  # 65
print(ord("á"))  # 225

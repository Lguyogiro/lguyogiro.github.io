Cadenas de Texto en Python

En este documento, exploraremos varias operaciones comunes que se pueden realizar con cadenas de texto en Python.

Strip

El método strip se utiliza para eliminar espacios en blanco al principio y al final de una cadena. Por ejemplo:


cadena = "   Hola, mundo!   "
cadena_limpia = cadena.strip(" ")
print(cadena_limpia)  # Hola, mundo!

Split

El método split se utiliza para dividir una cadena en una lista de subcadenas. Por ejemplo:


cadena = "Hola, mundo! Bienvenidos a Python."
partes = cadena.split()
print(partes)  # ['Hola,', 'mundo!', 'Bienvenidos', 'a', 'Python.']

Podemos especificar un delimitador para dividir la cadena, y también cuántas divisiones queremos hacer:


cadena = "Hola, mundo, Bienvenidos, a Python."
partes = cadena.split(",")  # ['Hola', ' mundo', ' Bienvenidos', ' a Python.']
print(partes)

En este caso, la cadena se divide en cuatro partes, utilizando la coma como delimitador.

Si solo queremos hacer una sóla división, podemos especificar el número de divisiones:


partes = cadena.split("!", 1)
print(partes)  # ['Hola, mundo', ' Bienvenidos! a Python.']

    
Count

    
El método count se utiliza para contar el número de veces que aparece una subcadena en una cadena. Por ejemplo:

    

cadena = "Hola, mundo! Hola a todos!"
conteo = cadena.count("Hola")
print(conteo)  # 2
    


    
Index

    
El método index se utiliza para encontrar la posición de una subcadena en una cadena. Por ejemplo:

    

cadena = "Hola, mundo!"
posicion = cadena.index("mundo")
print(posicion)  # 6
    


    
Lower/Upper/Title

    
Los métodos lower, upper y title se utilizan para cambiar el caso de las letras en una cadena. Por ejemplo:

    

cadena = "Hola, Mundo!"

# Convertir a minúsculas
minusculas = cadena.lower()
print(minusculas)  # hola, mundo!

# Convertir a mayúsculas
mayusculas = cadena.upper()
print(mayusculas)  # HOLA, MUNDO!

# Convertir a título
titulo = cadena.title()
print(titulo)  # Hola, Mundo!
    

    
Prabar checar si una cadena tiene puras mayúsculas, puras minúsculas, o si tiene la primera letra en mayúscula y el resto en minúsculas, podemos usar los métodos isupper, islower y istitle, respectivamente.

    

cadena = "HOLA, MUNDO!"
print(cadena.isupper())  # True

cadena = "hola, mundo!"
print(cadena.islower())  # True

cadena = "Hola, mundo!"
print(cadena.istitle())  # True
    



Startswith/Endswith


Los métodos startswith y endswith se utilizan para verificar si una cadena comienza o termina con una subcadena específica. Por ejemplo:



cadena = "Hola, mundo!"
print(cadena.startswith("Hola"))  # True
print(cadena.endswith("mundo!"))  # True

Las funciones

Las funciones son bloques de código que se pueden reutilizar en un programa. En Python, se definen con la palabra clave def:

def saludar():
    print("¡Hola, mundo!")

Para llamar a una función, simplemente escribimos su nombre seguido de paréntesis:

saludar()

Las funciones pueden recibir argumentos, que son valores que se pasan a la función para que los utilice. Por ejemplo:

def saludar(nombre):
    print("¡Hola", nombre, "!")

Para llamar a esta función, tenemos que pasarle un argumento:

saludar("Juan")

Las funciones pueden devolver un valor con la palabra clave return. Por ejemplo:

def sumar(a, b):
    return a + b

Para utilizar el valor devuelto por la función, lo asignamos a una variable:

resultado = sumar(3, 5)
print(resultado)  # Salida: 8

Las funciones también pueden tener argumentos con valores por defecto. Por ejemplo:

def saludar(nombre="mundo"):
    print("¡Hola", nombre, "!")

Si no pasamos un argumento a la función, se utilizará el valor por defecto:

saludar()  # Salida: ¡Hola mundo!

Ejercicios

  1. Escribe una función que tome dos números como argumentos y devuelva su producto.
  2. Escribe una función que tome una lista de números como argumento y devuelva la suma de todos los números en la lista.
  3. Escribe una función que tome una cadena como argumento y devuelva la cadena en mayúsculas.
  4. Escribe una función que tome una lista de cadenas como argumento y devuelva una lista con las cadenas en mayúsculas.
  5. Qué alguien nos de una lista de reglas para normalizar transcripciones. Escribe una función para realizar la normalización, o sea que tome como argumento una transcripción no normalizada, y devuelva la normalizada.
  6. Escribe una función que tome como su argumento el path de un archivo de texto, cree un diccionario de las frecuencias de las palabras, y devuelva la palabra mas frecuente.

Importar Módulos y Funciones en Python

En Python, los módulos son archivos que contienen código Python. Estos módulos pueden incluir funciones, clases y variables que puedes reutilizar en otros programas. Importar módulos y funciones te permite organizar tu código y evitar la repetición.

Importar un Módulo Completo

Para importar un módulo completo, utiliza la palabra clave import seguida del nombre del módulo. Por ejemplo:

import math

Esto te permite acceder a todas las funciones y variables definidas en el módulo math.

Importar Funciones Específicas

Si solo necesitas una función específica de un módulo, puedes importarla directamente usando la palabra clave from seguida del nombre del módulo y la palabra clave import seguida del nombre de la función. Por ejemplo:

from math import sqrt

Esto te permite usar la función sqrt directamente sin tener que escribir math.sqrt.

Importar con Alias

También puedes importar módulos o funciones con un alias para simplificar su uso. Utiliza la palabra clave as para asignar un alias. Por ejemplo:

import math as m

Ahora puedes usar m en lugar de math para acceder a las funciones del módulo.

from math import sqrt as raiz

Esto te permite usar raiz en lugar de sqrt.

La Biblioteca Estándar (Standard Library) de Python

Python viene con una extensa biblioteca estándar que incluye módulos y paquetes para realizar una amplia variedad de tareas. Algunos de los módulos más útiles y comunes incluyen:

re

El módulo re proporciona soporte para expresiones regulares, lo que permite buscar y manipular cadenas de texto. Por ejemplo:

import re
pattern = r'\b\w{4}\b'
text = 'Este es un ejemplo de texto con palabras de cuatro letras.'
matches = re.findall(pattern, text)
print(matches)  # Imprime las palabras de cuatro letras

string

El módulo string incluye constantes y clases para manipular cadenas de texto. Por ejemplo:

import string
print(string.ascii_letters)  # Imprime todas las letras ASCII

textwrap

El módulo textwrap proporciona funciones para dar formato a párrafos de texto. Por ejemplo:

import textwrap
text = "Este es un ejemplo de un párrafo que necesita ser envuelto."
wrapped_text = textwrap.fill(text, width=40)
print(wrapped_text)  # Imprime el texto envuelto a 40 caracteres por línea

difflib

El módulo difflib proporciona herramientas para comparar secuencias, incluyendo cadenas de texto. Por ejemplo:

import difflib
text1 = "Hola mundo"
text2 = "Hola Mundo"
differ = difflib.Differ()
diff = list(differ.compare(text1.split(), text2.split()))
print('\n'.join(diff))  # Imprime las diferencias entre las dos cadenas

unicodedata

El módulo unicodedata permite trabajar con datos de caracteres Unicode. Por ejemplo:

import unicodedata
char = 'ñ'
name = unicodedata.name(char)
print(name)  # Imprime el nombre del carácter Unicode

Estos son solo algunos ejemplos de los muchos módulos disponibles en la biblioteca estándar de Python. Explorar y aprender a usar estos módulos puede ayudarte a escribir código más poderoso y eficiente.

Cómo usar dir y help

Para explorar los módulos y funciones disponibles en Python, puedes usar las funciones integradas dir y help. Por ejemplo:

import re
print(dir(re))  # Imprime una lista de funciones y variables en el módulo re
print(help(re.search))  # Imprime la documentación de la función search en el módulo re

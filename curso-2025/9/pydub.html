Introducción a PyDub

PyDub es una biblioteca de Python que permite trabajar con archivos de audio de manera sencilla y intuitiva.

Para instalarlo, usa pip: pip3 install pydub

Convertir un archivo MP3 a WAV

PyDub permite convertir un archivo MP3 a WAV utilizando el método export() con el parámetro format. Por ejemplo:

from pydub import AudioSegment

audio = AudioSegment.from_file("audio.mp3")
audio.export("salida.wav", format="wav")

Esto creará un nuevo archivo llamado salida.wav con el mismo audio pero en formato WAV.

Cambiar la tasa de muestreo

La tasa de muestreo se refiere al número de muestras por segundo que se toman para representar un sonido. PyDub permite cambiar la tasa de muestreo utilizando el método .set_frame_rate()

from pydub import AudioSegment

audio = AudioSegment.from_file("audio.wav")
audio_with_altered_frame_rate = audio.set_frame_rate(44100)
audio_with_altered_frame_rate.export("salida.wav", format="wav")

Esto creará un nuevo archivo llamado salida.wav con el mismo audio pero con una tasa de muestreo de 44.1 kHz.

Reducir el número de canales

PyDub permite reducir el número de canales (de estéreo a mono) utilizando el método set_channels(). Por ejemplo:

from pydub import AudioSegment

audio = AudioSegment.from_file("audio_estereo.wav")
audio_mono = audio.set_channels(1)
audio_mono.export("salida_mono.wav", format="wav")

Esto creará un nuevo archivo llamado salida_mono.wav con el mismo audio pero en mono.

Cortar un trozo de audio según marcas de tiempo

PyDub permite cortar un trozo de audio según marcas de tiempo utilizando slicing. Por ejemplo, si tenemos un archivo de audio llamado audio.wav y queremos cortar desde los 10 segundos hasta los 20 segundos, podemos utilizar el siguiente código:

from pydub import AudioSegment

audio = AudioSegment.from_file("audio.wav")
audio_cortado = audio[10000:20000]  # en milisegundos
audio_cortada.export("salida_cortada.wav", format="wav")

Esto creará un nuevo archivo llamado salida_cortada.wav con el audio cortado desde los 10 segundos hasta los 20 segundos.

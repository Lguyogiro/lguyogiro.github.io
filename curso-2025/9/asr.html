Reconocimiento de voz / transcripción automática

Ahora vamos a ver cómo hacer reconocimiento de voz con Python y modelos abiertos de huggingface. Para esto, primero necesitamos instalar el paquete `transformers`:

    pip3 install transformers

Usando el siguiente código, podemos generar una transcripción automática para un audio.

from transformers import pipeline
model_name = "clu-ling/whisper-small-spanish"
asr_pipeline = pipeline("automatic-speech-recognition", model=model_name)
result = asr_pipeline("salida_16k_10-20.wav")
print(result["text"])

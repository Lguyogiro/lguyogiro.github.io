Ya han aprendido sobre los bucles en Python. Ahora, vamos a ver una forma más concisa y elegante de crear listas: list comprehensions.

¿Qué son las list comprehensions?

Las list comprehensions son una forma concisa de crear listas en Python. Son muy útiles cuando necesitas crear una lista basada en otra lista o secuencia.

La sintaxis de una list comprehension es la siguiente:

        
        nueva_lista = [expresion for elemento in secuencia]

Aquí la "expresión" puede realizar operaciones y/o transformaciones sobre el elemento (aquí `elemento` es la variable de iteración que tú defines para referirte a cada elemento en la lista).

Por ejemplo, si tienes una lista de números y quieres crear una nueva lista con el cuadrado de cada número, puedes hacerlo con una list comprehension.

        
        numeros = [1, 2, 3, 4, 5]
        cuadrados = [x ** 2 for x in numeros]
        print(cuadrados)

También podemos incluir condiciones en las list comprehensions. Por ejemplo, si queremos crear una lista con los números pares de una lista original, podemos hacerlo de la siguiente manera:

        
        numeros = [1, 2, 3, 4, 5]
        pares = [x for x in numeros if x % 2 == 0]
        print(pares)

Convertir un bucle for en una list comprehension

El siguiente bucle for:

        
        cuadrados = []
        for x in range(1, 6):
            cuadrados.append(x ** 2)
        print(cuadrados)

Se puede reescribir como una list comprehension:

        
        cuadrados = [x ** 2 for x in range(1, 6)]
        print(cuadrados)

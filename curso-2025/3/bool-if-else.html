Expresiones booleanas

  • Las expresiones booleanas son de suma importancia en la programación. Son básicamente preguntas que pueden tener una de dos respuestas: True (verdad) o False (falso).
  • Típicamente, tienen un valor a la izquierda, un valor a la derecha, y en medio un operador lógico o matemático
    • ==: Igual a
    • !=: No igual a
    • >: Mayor que
    • <: Menor que
    • >=: Mayor o igual que
    • <=: Menor o igual que
  • Por ejemplo: 
    print("this" == "that") # False
    print((10*10) > 99) # True
    print("hola"[0] != "a") # True
  • Operadores lógicos comunes incluyen and, or y not.
  • Ejemplo de operadores lógicos: 
    a = True
b = False
print(a and b)  # False
print(a or b)   # True
print(not a)    # False

Condicionales

Los condicionales son estructuras de control que permiten tomar decisiones en un programa. En Python, los condicionales se implementan con la estructura if, elif y else.

if condicion:
    # Código a ejecutar si la condición es verdadera
elif otra_condicion:  # Esta línea es opcional, sólo se usa si quieres probar mas de una condición.
    # Código a ejecutar si la condición anterior es falsa y esta es verdadera
else:
    # Código a ejecutar si todas las condiciones anteriores son falsas

Por ejemplo:

edad = 18
if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

Con listas y diccionarios

También podemos evaluar expresiones booleanas que usan list, set, o dict. Por ejemplo, para saber si una lista contiene un elemento, usamos ELEMENTO in LIST


    numeros = [3, 4, 12, 89, 100, 39, 101, 2, 1, 1789, 1988, 20]
    print(5 in numeros) # False
    print(101 in numeros) # True

Igual, para averiguar si una clave existe en un diccionario, usamos la misma expresión

edades = {"Juan": 25, "María": 30, "Pedro": 18, "Ana": 20} print("Juan" in edades) # True print("Luis" in edades) # False if "Juan" in edades: print("Juan tiene " + str(edades["Juan"]) + " años") else: print("No tengo la edad de Juan")

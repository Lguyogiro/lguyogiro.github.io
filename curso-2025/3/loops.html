Loops (Bucles) en Python

Los bucles son estructuras de control que permiten repetir un bloque de código varias veces. En Python, hay dos tipos de bucles: for y while.

Bucle for

El bucle for se utiliza para iterar sobre una secuencia (como una lista, tupla, diccionario, o cadena) y ejecutar un bloque de código una vez para cada elemento de la secuencia. Por ejemplo:

        
        colores = ["rojo", "verde", "azul"]
        for color in colores:
            print(color)

En este caso, el bucle for itera sobre la lista colores e imprime cada color en una línea separada.

También se puede iterar sobre un string:

        
        for letra in "hola":
            print(letra)

Bucle while

El bucle while se utiliza para repetir un bloque de código mientras una condición sea verdadera. Por ejemplo:

        
        i = 1
        while i < 6:
            print(i)
            i += 1

En este caso, el bucle while imprime los números del 1 al 5.

Es importante tener cuidado con los bucles while, ya que si la condición nunca se vuelve falsa, el bucle se ejecutará para siempre.

continue y break en los bucles

En Python, puedes usar las palabras clave continue y break para controlar el flujo de los bucles.

continue se usa para saltar a la siguiente iteración del bucle, mientras que break se usa para salir del bucle por completo.

Por ejemplo:

    
    mi_contraseña = "1234"
    intentos = ["H3llo", "root", "1234", "password", "admin"]
    for intento in intentos:
        if intento == mi_contraseña:
            print("Contraseña correcta!")
            break
        else:
            continue

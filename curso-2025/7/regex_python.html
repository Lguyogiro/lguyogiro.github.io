Para utilizar las expresiones regulares en un programa de Python, tenemos que importar el módulo re:

import re

Hay varias funciones del módulo `re`, como search, match, sub, split, findall, y finditer

  • search(patron, cadena): Busca la primera ocurrencia del patrón en la cadena y devuelve un objeto de coincidencia si lo encuentra. Si no se encuentra ninguna coincidencia, devuelve None. Ejemplo: 
    import re
cadena = "Hola, ¿cómo estás?"
patron = "cómo"
resultado = re.search(patron, cadena)
if resultado:
    print(f"Se encontró '{patron}' en la posición {resultado.start()}")
else:
    print(f"No se encontró '{patron}'")
  • match(patron, cadena): Busca el patrón al principio de la cadena y devuelve un objeto de coincidencia si lo encuentra. Si no se encuentra ninguna coincidencia, devuelve None. Ejemplo: 
    import re
cadena = "Hola, mundo"
patron = "Hola"
resultado = re.match(patron, cadena)
if resultado:
    print(f"La cadena comienza con '{patron}'")
else:
    print(f"La cadena no comienza con '{patron}'")
  • sub(patron, reemplazo, cadena): Reemplaza todas las ocurrencias del patrón en la cadena por el texto de reemplazo. Ejemplo: 
    import re
cadena = "Me gusta el café"
patron = "café"
reemplazo = "té"
nueva_cadena = re.sub(patron, reemplazo, cadena)
print(nueva_cadena)  # Salida: Me gusta el té
  • split(patron, cadena): Divide la cadena en una lista de substrings según el patrón. Ejemplo: 
    import re
cadena = "hola,mundo,adiós"
patron = ","
lista = re.split(patron, cadena)
print(lista)  # Salida: ['hola', 'mundo', 'adiós']
  • findall(patron, cadena): Busca todas las ocurrencias del patrón en la cadena y devuelve una lista de coincidencias. Ejemplo: 
    import re
cadena = "Mi número de teléfono es 56 1234 7890 y el de mi amigo es 797 1717 8982"
patron = r"\d{2,3} \d{4} \d{4}"
numeros = re.findall(patron, cadena)
print(numeros)  # Salida: ['56 1234 7890', '797 1717 8982']
  • finditer(patron, cadena): Busca todas las ocurrencias del patrón en la cadena y devuelve un iterador de objetos de coincidencia. Ejemplo: 
    import re
cadena = "Mi número de teléfono es 56 1234 7890 y el de mi amigo es 797 1717 8982"
patron = r"\d{2,3} \d{4} \d{4}"
numeros = re.findall(patron, cadena)
for coincidencia in re.finditer(patron, cadena):
    print(f"Se encontró un número de teléfono en la posición {coincidencia.start()}: {coincidencia.group()}")
